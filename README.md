# Сапер (Minesweeper) — Курсовий проект учня 24 групи Бовшика Андрія

## Про проект

Цей проект — консольна версія гри Сапер, реалізована на C.  
Гравець повинен відкривати клітинки на полі розміром 10×10, уникати мін і намагатися відкрити всі безпечні клітинки.

Гра демонструє:
- роботу з масивами та двовимірними структурами
- використання циклів і умов
- базову роботу з введенням/виведенням в консоль
